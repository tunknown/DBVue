  • Ограничение текущей реализации- поле P должно быть только одно?

  • TColumn.ButtonStyle определяется из типа лукапа и типа данных.

  • S|PaymentItem деталь для L|Payment, но не для D|Item, т.к. Payment- журнал, а Item- справочник, хотя оба они- FK. Для R|PaymentItem их различие не важно и их можно использовать, как детали.
